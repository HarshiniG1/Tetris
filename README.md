---

Option 2 — Run with Docker  
Make sure Docker is installed, then run:

bash  
# Pull and run in one command  
docker run -p 8080:80 nginx:alpine  

Or build it yourself from the source:

Step 1 — Clone the repo  
bash  
git clone https://github.com/YOUR_USERNAME/kawaii-tetris.git  
cd kawaii-tetris  

Step 2 — Build the Docker image  
bash  
docker build -t kawaii-tetris .  

Step 3 — Run the container  
bash  
docker run -p 8080:80 kawaii-tetris  

Step 4 — Open your browser  
http://localhost:8080  

💡 To stop the container, press Ctrl + C or run docker ps to find the container ID, then docker stop <ID>

🎮 How to Play

Control | Action
← → Arrow Keys | Move block left / right  
↑ Arrow Key | Rotate block  
↓ Arrow Key | Soft drop (faster fall)  
Space | Hard drop (instant)  
P | Pause / Resume  
Tap (mobile) | Rotate block  
Swipe left/right | Move block  
Swipe down | Hard drop
